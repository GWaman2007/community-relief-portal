# Community Relief Portal

## Project Tone & Terminology
This application is a **Disaster Response & Volunteer Coordination Platform**. It is designed to be used by verified NGOs, community volunteers, and system administrators. 

When writing copy, tooltips, or documentation, DO NOT use overly militaristic, espionage, or "spy" terminology (e.g., avoid "Operatives", "Central Command", "Gatekeeper", "Kill Switch"). 

Instead, use standard humanitarian and community-centric language:
- **Admin**: System Administrator, Relief Coordinator
- **NGO**: Verified Partner, NGO Contributor
- **Volunteer**: Registered Volunteer, Community Responder
- **Actions**: Authorize, Dispatch, Resolve, Verify, Suspend

## Aesthetic Guidelines
The UI utilizes a modern **dark-mode glassmorphic aesthetic** with high-fidelity components:
- Backgrounds: `bg-background` with subtle circular radial gradients
- Cards: `bg-card/60 backdrop-blur-xl border border-border/50`
- Icons: `lucide-react`
- Typography: Modern, readable font with `oklch` standard colors.
